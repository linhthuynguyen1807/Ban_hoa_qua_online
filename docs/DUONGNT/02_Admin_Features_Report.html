Báo cáo Cập nhật: Admin Features

Phase 1 & 2 - Hoàn Thành

Thành viên thực hiện: Duong (DUONGNTHE191504)

Báo cáo này tổng hợp các tính năng dành cho Quản trị viên (Admin) đã được phát triển trong dự án Bán Hoa Quả Online.

1. Phase 1: Quản lý Người dùng (User Management)

Chức năng xem danh sách và khóa/mở khóa tài khoản khách hàng, chủ shop.

  • Database & DAO: Cập nhật hàm findAll() và updateUserStatus() trong UserDAO.java.
  • Service: Khởi tạo UserService.java để chứa logic nghiệp vụ.
  • API & Servlet:
    • Tạo AdminUserServlet.java (Ánh xạ: /admin/users) để hiển thị giao diện.
    • Tạo AdminUserStatusAPI.java (Ánh xạ: /admin/users/status) xử lý AJAX POST khóa/mở khóa tài khoản.
  • Giao diện (JSP):
    • Tạo admin-sidebar.jsp làm thanh điều hướng dùng chung cho Admin.
    • Tạo user-list.jsp: Hiển thị danh sách, tìm kiếm trực tiếp (Client-side), và gọi AJAX kèm Token CSRF.

2. Phase 2: Phê duyệt Cửa hàng (Shop Approvals)

Chức năng giúp Admin duyệt hoặc từ chối các yêu cầu mở cửa hàng của người dùng.

  • Database & DAO: Đã có sẵn các hàm trong ShopProfileDAO.java.
  • Service: Tạo mới ShopService.java để gọi DAO lấy danh sách chờ duyệt.
  • API & Servlet:
    • Cập nhật ShopApprovalServlet.java (/admin/shops) để lấy dữ liệu.
    • Tạo ShopApprovalAPI.java (/admin/shops/approve) để xử lý Approve/Reject qua AJAX.
  • Giao diện (JSP):
    • Tạo shop-approvals.jsp: Hiển thị danh sách yêu cầu. Nút Duyệt (cấp quyền ngay) và Từ chối (hiển thị Popup nhập lý do từ chối).

3. Phase 2: Kiểm duyệt Đánh giá (Review Moderation)

Chức năng ẩn/hiện các đánh giá sản phẩm nhằm kiểm soát spam hoặc nội dung không phù hợp.

  • Database & DAO:
    • Thêm trường productName vào model Review.java.
    • Cập nhật ReviewDAO.java: Thêm hàm findAllForAdmin() JOIN 4 bảng để lấy tên sản phẩm & người dùng, và hàm updateVisibility().
  • Service: Cập nhật ReviewService.java để kết nối DAO mới.
  • API & Servlet:
    • Tạo AdminReviewServlet.java (/admin/reviews) lấy dữ liệu.
    • Tạo AdminReviewAPI.java (/admin/reviews/visibility) để xử lý bật/tắt (Toggle) qua AJAX.
  • Giao diện (JSP):
    • Tạo review-management.jsp: Bảng đánh giá chi tiết với công tắc (Toggle Switch) trực quan để Ẩn / Hiện.
