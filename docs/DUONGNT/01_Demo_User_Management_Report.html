2. Hướng dẫn Test cho Demo

  • Bước 1: Nếu chưa có database, bạn cần chạy lại file Setup_OnlineFruitShopping.sql trong SQL Server Management Studio để nạp mật khẩu mới.
  • Bước 2: Build và Run project trên Tomcat.
  • Bước 3: Truy cập vào http://localhost:8080/Ban_Hoa_Qua_Online/admin/users.
  • Bước 4: Khi hệ thống yêu cầu đăng nhập (giả sử bạn đã làm luồng Auth), hãy dùng email admin@fruitshop.local và mật khẩu 123456.
  • Bước 5: Tại trang Quản lý người dùng, bạn sẽ thấy danh sách dạng bảng (màu xanh lá tương đồng với màu thương hiệu MetaFruit).
  • Bước 6: Nhấn nút "Khóa" hoặc "Mở khóa" ở một User bất kỳ (trừ Admin). Bạn sẽ thấy thông báo nhỏ bật lên (Toast) và trạng thái đổi màu động mà không cần tải lại trang.
